# 验收标准

## 功能验收

### 1. 文本内容保存
- [ ] 在 Agent 栏输入文本后，刷新页面，文本内容被恢复
- [ ] 输入框在页面加载时自动填充之前保存的文本

### 2. 附件保存
- [ ] 添加附件后，刷新页面，附件列表被恢复
- [ ] 附件信息（文件名）正确显示

### 3. 清除功能
- [ ] 点击"清除"按钮后，输入框内容被清空
- [ ] 刷新页面后，内容保持清空状态（localStorage 已清除）

## 测试方法
1. 打开 /home 页面
2. 在 Agent 栏输入一些文本
3. 添加一个测试附件（如果有上传功能）
4. 刷新页面
5. 验证文本和附件是否恢复
6. 点击清除按钮，刷新页面，验证内容已清空

## 代码检查
- 使用 `git diff` 检查修改范围
- 确认只修改了前端文件（.js, .html）
- 确认没有意外变更
