时间管理大师

基于 Django + LangGraph 的开源智能日程管理项目

一个集 日程、提醒、待办、群组协作 与 AI Agent 对话助手 于一体的现代化日程管理系统。 本站点为项目的演示与学习交流平台，非商业服务。

Python 3.12 Django 5.x LangGraph MIT License 开源项目

项目特性

围绕个人时间管理打造的全功能开源平台

日程管理

支持单次与重复日程，完整 RRule（RFC 5545）规则：FREQ / INTERVAL / COUNT / UNTIL / BYDAY / EXDATE。

智能提醒

多优先级、延迟提醒、重复提醒，与日程、待办双向关联，支持桌面与浏览器通知。

待办事项

艾森豪威尔四象限法分类，支持依赖关系、预估时长，帮助理性安排优先级。

群组协作

多人共享日程，三级权限管理，版本号增量同步，避免冲突与覆盖。

AI Agent 助手

LangGraph 驱动，WebSocket 实时对话，支持自然语言创建与查询日程、附件 OCR、联网搜索。

iCalendar 互通

导入兼容标准 iCal；订阅 Feed 可直接添加到 Apple Calendar / Google Calendar，支持 CalDAV。

技术栈

现代、开放、可自部署

Django 5.x Django REST Framework Channels + Daphne LangGraph + LangChain FullCalendar 6.x Bootstrap 5 SQLite / PostgreSQL icalendar AES-256-GCM
