# 2026-05-23 Agent 前端 Token 可视化升级

## 变更摘要

- 上下文构建详情顶部新增 token 百分比堆积图，展示 input cache miss、input cache hit、output 三类 token。
- 移除上下文详情中的独立 `viz-section-token-stats`，token 统计改为请求摘要信息。
- 取消系统提示词默认展开，消息详情默认收起。

- 重构 AI 设置 - Token 统计，将旧表格改为模型分布、成本拆分、请求元数据、最近请求四类视图。
- 清理 `agent-config.js` 中旧版 `updateTokenStatsUI()` / `updateModelStatsTable()` 与新版同名覆盖问题。
- 更新 `home.html` 中 `agent-chat.css`、`agent-chat.js`、`agent-config.js` 静态版本号。

## 经验规则

- 前端 token 可视化应只读取三段 token：`input_cache_miss_tokens`、`input_cache_hit_tokens`、`output_tokens`。
- 请求级明细不直接展示 `diagnostics`，避免误露 provider/request 调试信息。
- 静态 JS/CSS 修改后必须同步更新 `home.html` 版本号并运行静态校验。
