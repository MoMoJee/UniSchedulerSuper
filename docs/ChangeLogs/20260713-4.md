# 2026-07-13 API 示例与 Planner V1 退役契约升级

## 接口

- Event、个人 Event Group、Todo、Reminder 与共享 occurrence 的示例全面切换至 `/api/v2/`。
- Planner V1 URL 统一返回 `410 Gone`、`planner_v1_api_retired` 和可机器读取的 replacement，不再让数据库防写异常泄漏成 500。
- V1 tombstone 请求不读取/写入 legacy 归档，也不写 normalized 实体；未认证请求仍先执行认证。
- Group/Todo/Reminder V2 HTTP 请求改为严格字段契约，未知或旧字段返回 `422 unsupported_field`，不再静默忽略。
- 重复 Event `scope=all` 与 Reminder 对齐：允许修改系列每日时刻，禁止移动系列起始日期，返回 `series_date_change_forbidden`。
- 旧共享组 `/api/share-groups/<id>/events/` 改为 410，并指向有界 V2 occurrences 查询。
