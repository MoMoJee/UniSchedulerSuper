# FR-4B Todo 写入验收报告

> 验收日期：2026-07-14  
> 结论：**通过（能力边界已明确）**。

## 已交付与能力矩阵

| 能力 | 状态 | 说明 |
| --- | --- | --- |
| 创建/编辑/删除 | 支持 | V2 Todo create/PATCH/DELETE，写入版本来自当前查询结果。 |
| 完成/恢复、状态筛选 | 支持 | `pending/completed` 走 V2，成功后失效 Todo 与日历投影。 |
| 描述、截止时间、重要性、紧急度 | 支持 | 表单发送 V2 已允许字段。 |
| 真实 Todo→Event 转换 | 支持 | 调用 `/api/v2/todos/<id>/convert/`；不会用前端复制冒充转换。 |
| 手动排序/跨区拖拽 | 明确不支持 | 当前 V2 Todo 模型没有排序位置字段；不提供刷新后回退的假拖拽。仍按服务端截止时间排序。 |
| Todo 附件/共享/Agent 上下文卡片 | FR-5 | 当前 Todo V2 command 未定义附件/共享写字段；统一资源引用必须和 Agent/文件选择器一起迁移。 |

## 测试结果

Playwright 覆盖创建和真实转换，网络断言两个请求均只到 `/api/v2/todos/...`，转换 body 含 `expected_version` 且原 Todo 字段映射到 Event payload。单元/API 测试覆盖版本校验和错误转换；全量前端质量门通过。
