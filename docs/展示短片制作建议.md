# UniSchedulerSuper 展示短片制作建议

> 目标：制作一个 2~3 分钟的短片，有力呈现 UniSchedulerSuper 相较于传统日程表的核心创新点（AI、协作、规划），同时尽量降低制作成本和时间投入。

---

## 一、整体策略：省时省力的制作路径

考虑到你并非专业动画制作者，推荐以下 **三条可选路径**，难度从低到高排列：

| 路径 | 工具 | 预计耗时 | 效果 |
|------|------|---------|------|
| ✅ **路径 A（推荐）** | AI 视频生成 + PPT/录屏 | 2~4 小时 | 专业感强，适合演示 |
| 🔵 **路径 B** | Canva / 剪映 + AI 配音 | 1~2 小时 | 简洁清晰，适合社交分享 |
| 🟡 **路径 C（最简）** | 纯 PPT + 屏幕录制 | 0.5~1 小时 | 朴素但信息完整 |

---

## 二、推荐路径 A：AI 辅助短片（2~4 小时）

### 2.1 工具组合

| 用途 | 工具 | 说明 |
|------|------|------|
| **脚本生成** | ChatGPT / Claude | 提供项目功能，让 AI 写旁白脚本 |
| **PPT / 幻灯片** | Gamma.app | 输入大纲，AI 自动生成带动画的幻灯片 |
| **动画演示** | 屏幕录制 (OBS / Windows 自带) | 录制项目真实操作界面 |
| **AI 配音** | 微软 Edge TTS / 剪映 AI 配音 | 自动生成中文旁白，不需要自己录音 |
| **视频剪辑** | 剪映（PC 版）/ CapCut | 拼合素材、添加字幕、加背景音乐 |

### 2.2 制作步骤

#### 第一步：让 AI 写脚本（15 分钟）

把下面这段 Prompt 发给 ChatGPT 或 Claude：

```
我有一个项目叫 UniSchedulerSuper，是一个基于 Django + LangGraph 的智能日程管理系统。
核心创新点：
1. AI 智能助手：通过自然语言对话创建/查询日程，例如"帮我安排下周二下午两点和 John 开会"
2. 多用户协作：共享日历、3 级权限管理、增量版本同步
3. 艾森豪威尔四象限：任务自动分类（重要/紧急维度）
4. 全平台日历订阅：标准 iCalendar 格式，兼容苹果/谷歌/Outlook
5. 语音输入、OCR 图片识别、附件解析（PDF/Word/Excel）
6. 版本回滚：可恢复历史版本的日程数据
7. MCP 集成：支持 Claude Desktop 直接操控日程

请帮我写一个 2~3 分钟的演示短片旁白脚本，要突出 AI 与传统日程表的对比，
有吸引力，语气轻松，适合 B 站或微信视频号发布。
```

#### 第二步：用 Gamma.app 生成幻灯片（20 分钟）

1. 打开 [https://gamma.app](https://gamma.app)（免费账号即可）
2. 选择 "Create with AI"
3. 输入你的大纲（见下方第三节"内容大纲"）
4. 选择合适的主题风格（推荐：科技/深色系）
5. 导出为 PDF 或直接在线演示

#### 第三步：录制真实操作演示（30~60 分钟）

按以下顺序录制 3~5 个关键操作片段（每段 20~40 秒）：

1. **AI 对话创建日程**：打开 `/agent/` 页面，输入自然语言，展示 AI 返回结果
2. **日历视图**：展示 FullCalendar 界面，周/月视图切换
3. **四象限任务管理**：展示 Todo 的重要/紧急分类
4. **协作共享**：展示创建共享群组、添加成员
5. **语音/OCR 输入**：上传一张有日程信息的图片，展示 AI 自动解析

> 💡 录制技巧：提前准备好演示数据，关闭通知，用 1080p 全屏录制，操作要慢一点清晰展示

#### 第四步：AI 配音（15 分钟）

- 使用 **剪映** PC 版：粘贴脚本文字 → "AI 朗读" → 选择中文发音人（推荐：专业男声/女声）
- 或使用 **微软 TTS（免费）**：在 PowerShell 中用 `Invoke-RestMethod` 调用，或访问 [https://tts.microsoft.com](https://azure.microsoft.com/zh-cn/products/ai-services/text-to-speech)

#### 第五步：剪辑合成（30~60 分钟）

在 **剪映（PC）** 中：
1. 导入幻灯片截图/录屏片段
2. 对齐 AI 配音音轨
3. 添加字幕（剪映支持自动识别字幕）
4. 加入背景音乐（剪映免费音乐库）
5. 添加简单过渡动画（淡入淡出即可）
6. 导出 1080p MP4

## 三、短片内容大纲

建议按以下结构组织，控制在 2~3 分钟：

### 开场（0:00 ~ 0:20）—— 痛点引入

> **画面**：传统日程表截图（Google Calendar / 纸质日历）
> **旁白**："你有没有这样的烦恼——想安排一个会议，要手动填写时间、地点、提醒……遇到重复事件更是麻烦。而且，你的日程只有你自己能看，无法和团队高效协作。"

### 第一幕（0:20 ~ 0:50）—— AI 对话创建日程

> **画面**：UniSchedulerSuper 的 AI 对话界面录屏
> **旁白**："UniSchedulerSuper 让你用说话的方式管理日程。只需告诉 AI 助手你的需求，它就能自动帮你创建、修改、查询日程，甚至理解模糊的时间表达。"
> **演示**：输入"帮我把明天下午三点的团队会议推迟一小时"，展示 AI 实时响应

### 第二幕（0:50 ~ 1:20）—— 智能规划：四象限任务管理

> **画面**：Todo 四象限视图
> **旁白**："不只是安排时间，还帮你做决策。系统内置艾森豪威尔四象限，自动将你的任务按重要性和紧急度分类，让你清楚知道该先做什么。"

### 第三幕（1:20 ~ 1:50）—— 多人协作

> **画面**：共享日历、权限管理界面
> **旁白**："团队协作也不再困难。创建共享日历，邀请成员加入，设置查看或编辑权限，所有变更实时同步，再也不用群里发截图对日程了。"

### 第四幕（1:50 ~ 2:20）—— 全平台兼容 + 语音/OCR

> **画面**：iCal 订阅链接 + Apple Calendar 截图；语音输入 / 图片上传演示
> **旁白**："生成标准 iCalendar 订阅链接，直接同步到你的手机日历。还支持语音输入和图片识别——拍一张会议通知，AI 自动帮你提取时间地点，一键添加到日程。"

### 结尾（2:20 ~ 2:40）—— 总结 + 项目信息

> **画面**：Logo + GitHub 地址
> **旁白**："UniSchedulerSuper，不只是日程表，更是你的智能时间管理助手。开源免费，欢迎体验和贡献。"
> **字幕**：GitHub: github.com/MoMoJee/UniSchedulerSuper

## 四、备选路径 B：Canva + 剪映（1~2 小时，更简单）

1. 打开 [https://www.canva.com](https://www.canva.com) → 搜索"科技产品演示"模板
2. 按上方内容大纲替换文字和截图
3. 导出为视频（Canva 支持直接导出 MP4 并添加简单动画）
4. 在剪映中加 AI 配音和字幕
5. 完成导出

> ✅ Canva 免费版完全够用，导出 MP4 需要 Canva Pro 或直接导出 GIF/图片后在剪映合成

## 五、备选路径 C：纯 PPT（0.5~1 小时，最快）

用 PowerPoint 或 WPS 制作如下幻灯片（每页用"切换动画"增加过渡效果）：

| 页码 | 内容 |
|------|------|
| 1 | 标题页：UniSchedulerSuper + 副标题"让 AI 管理你的时间" |
| 2 | 传统日程表的痛点（3 条 bullet points + 配图） |
| 3 | 核心特性一览（图标 + 简短说明） |
| 4 | AI 对话演示截图（配说明文字） |
| 5 | 四象限规划截图（配说明文字） |
| 6 | 多人协作截图（配说明文字） |
| 7 | iCal 订阅 + 语音/OCR 截图 |
| 8 | 结尾页：GitHub 地址 + 联系方式 |

录屏 PPT 播放过程 + 自己录音旁白 = 完成。

## 六、实用 AI Prompt 模板

### 让 AI 生成短片脚本

```
请为以下产品写一个 2 分钟展示视频的旁白脚本：

产品名：UniSchedulerSuper
核心卖点（区别于传统日程表）：
- AI 自然语言对话，直接说话安排日程
- 艾森豪威尔四象限任务管理
- 多人共享日历和协作
- 语音输入 + 图片OCR识别
- 标准iCalendar跨平台同步

受众：大学生、职场人、技术爱好者
风格：轻松、有感染力，突出"智能"和"省时"
语言：中文
时长控制：约 400~500 字
```

### 让 AI 生成 PPT 大纲（用于 Gamma.app）

```
帮我生成一个 8 页的 PPT 大纲，用于展示 UniSchedulerSuper 项目。
要求：
- 第 1 页：标题和副标题
- 第 2 页：传统日程表的 3 个痛点
- 第 3-7 页：5 个核心功能，每页一个（AI对话/四象限/协作/语音OCR/全平台同步）
- 第 8 页：总结 + GitHub 地址
每页包含：标题、2~3 条说明、建议配图描述
```

## 七、素材准备清单

在录屏/截图之前，建议先在本地准备好以下演示数据：

- [ ] 创建至少 5 个示例事件（包括重复事件）
- [ ] 创建 3~5 个不同优先级的 Todo
- [ ] 创建一个共享日历群组并添加测试账号
- [ ] 准备一张含有日程信息的图片（用于 OCR 演示）
- [ ] 配置好 AI API Key（DeepSeek 或 OpenAI，用于 Agent 演示）
- [ ] 在 iPhone/Android 上设置好 iCal 订阅链接

## 九、最终建议

> 🎯 **如果时间只有 1 小时**：选路径 C，用 PPT + 屏幕录制快速完成，内容优先于形式。
>
> 🎯 **如果有 2~3 小时**：选路径 A，用 Gamma.app 生成美观幻灯片 + 剪映 AI 配音，效果专业。
>
> 🎯 **核心原则**：真实的产品录屏演示 > 任何动画效果。让观众看到 AI 真实响应的过程，比任何特效都有说服力。

**最重要的一个镜头**：展示在 AI 对话框里输入一句自然语言，然后日历上实时出现对应的事件——这一个画面就能说明一切。
